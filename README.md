# Customer Behavior Data Analysis Project

An end-to-end **Data Analyst portfolio project** that simulates a real-world business analytics workflow using **Python, SQL, and Power BI**. This project demonstrates how raw customer transaction data can be transformed into **actionable business insights** through data preparation, analysis, visualization, and reporting.

## Project Objective

The goal of this project is to replicate a **corporate-style data analytics pipeline**, showing the ability to:

* Clean and prepare raw business data
* Perform exploratory data analysis
* Store and query data using SQL
* Build dashboards for stakeholders
* Present insights and recommendations

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL (PostgreSQL / MySQL / SQL Server)
* Power BI
* Jupyter Notebook
* Git & GitHub

## Project Workflow

### 1. Data Preparation & EDA (Python)

* Import dataset
* Handle missing values
* Data transformation
* Exploratory data analysis
* Feature understanding

### 2. Database & SQL Analysis

* Create SQL database
* Load cleaned data from Python
* Write business queries
* Customer segmentation analysis
* Purchase behavior insights

### 3. Dashboard Development (Power BI)

Build an interactive dashboard to visualize:

* Sales trends
* Customer segments
* Product performance
* Revenue insights
* Category-wise analysis


## Key Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis
* SQL Querying
* Data Modeling
* Dashboard Design
